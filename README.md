# Strav Framework

**The Modern, AI-First Backend Framework for Bun** — Built from the ground up for exceptional performance, enterprise features, and developer experience.

## Features

- 🚀 **Bun-Native Performance** — Optimized for Bun's speed and efficiency
- 🧠 **AI-First Architecture** — Multi-provider AI integration with agents and tool use
- 🏢 **Enterprise Ready** — OAuth2 server, multi-tenant database, Stripe billing, full-text search
- 🎯 **Full-Stack Development** — Backend framework with Vue islands for modern web apps
- 📦 **Modular Design** — 20+ focused packages with clean separation of concerns
- 🛠️ **Exceptional DX** — Rich CLI tooling, code generators, and testing utilities

## Create Your First Application

```bash
# 1. Create a new web application
bunx @strav/spring my-app --web --db=my_app_db

# 2. Navigate to your project
cd my-app

# 3. Install dependencies
bun install

# 4. Start development server
bun run dev
```

Visit `http://localhost:3000` — you'll see your Strav application running with interactive Vue islands and user management.
